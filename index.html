FAQs

Is this a live application?

No — this is a static front-end demo. All data lives only in your browser.

Do I need an account?

No. Guest usage is allowed, but accounts enable saving and tracking.

Can I download invoices?

In the live application, yes — as A4 PDFs. That export feature is disabled in this static demo.

Supported currencies?
